Chapitre I – Les suites

Fiche résumée

Première Difficile

En mathématiques, une suite est une famille d'éléments — appelés ses termes — indexée par les entiers naturels. Ce cours va donc vous apprendre comment définir une suite et l'étudier. On y verra notamment l'étude du sens de variation, la représentation de suites dans le plan, et plus encore !

Retour au cours

Généralités

Définition

On appelle suite une fonction de N\mathbb{N}N dans R\mathbb{R}R : cette fonction va prendre des éléments de l’ensemble de départ N\mathbb{N}N et va les amener dans l’ensemble d’arrivée R\mathbb{R}R.

Définition

Il y a plusieurs manières de définir une suite :

  • Par récurrence : On donne le premier terme de la suite ainsi que le terme au rang n+1n+1n+1.

  • Par son terme général : On donne le nnn-ième terme de la suite en fonction de nnn.

Attention ! Bien que ces deux modes de génération soient les principaux, il en existe d’autres : algorithme, motifs géométriques, ...

Suites arithmétiques

Définition

Une suite (un)(u_n)(un​) est dite arithmétique si elle est de la forme un+1=un+ru_{n+1} = u_n + run+1​=un​+r avec r∈Rr \in \mathbb{R}r∈R.

Raison

Le réel rrr est la raison de la suite (si r>0r > 0r>0, (un)(u_n)(un​) est strictement croissante, si r<0r < 0r<0, (un)(u_n)(un​) est strictement décroissante et si r=0r = 0r=0, (un)(u_n)(un​) est constante).

Il est possible de trouver le terme général d’une suite arithmétique :

Terme général

On note ppp le rang initial de la suite (celui à partir duquel la suite est définie) :

un=up+(n−p)×ru_n = u_p + (n-p) \times run​=up​+(n−p)×r

Et si (un)(u_n)(un​) est définie à partir du rang 000 (on a p=0p = 0p=0) :

un=u0+(n−0)×r=u0+n×ru_n = u_0 + (n-0) \times r = u_0 + n \times run​=u0​+(n−0)×r=u0​+n×r

Somme des termes

1+2+⋯+n=n(n+1)2\displaystyle{1 + 2 + \dots + n = \frac{n(n + 1)}{2}}1+2+⋯+n=2n(n+1)​ pour tout n∈Nn \in \mathbb{N}n∈N.

Suites géométriques

Définition

Une suite (vn)(v_n)(vn​) est dite géométrique si elle est de la forme vn+1=vn×qv_{n+1} = v_n \times qvn+1​=vn​×q avec q∈Rq \in \mathbb{R}q∈R.

Raison

Le réel qqq est la raison de la suite (si q>1q > 1q>1, (vn)(v_n)(vn​) est strictement croissante, si 0<q<10 < q < 10<q<1, (vn)(v_n)(vn​) est strictement décroissante et si q=1q = 1q=1 ou 000, (vn)(v_n)(vn​) est constante).

Il est possible de trouver le terme général d’une suite géométrique :

Terme général

On note ppp le rang initial de la suite (celui à partir duquel la suite est définie) :

vn=vp×qn−pv_n = v_p \times q^{n-p}vn​=vp​×qn−p

Et si (vn)(v_n)(vn​) est définie à partir du rang 000 (on a p=0p = 0p=0) :

vn=v0×qn−0=v0×qnv_n = v_0 \times q^{n-0} = v_0 \times q^nvn​=v0​×qn−0=v0​×qn

Somme des termes

Soit n≠0n \neq 0n=0 un entier et qqq un réel, alors :

  • Si q≠1q \neq 1q=1, alors 1+q1+q2+⋯+qn=1−qn+11−q\displaystyle{1 + q^1 + q^2 + \dots + q^n = \frac{1 - q^{n + 1}}{1 - q}}1+q1+q2+⋯+qn=1−q1−qn+1​.

  • Si q=1q = 1q=1, alors 1+q1+q2+⋯+qn=1+1+1+⋯+1⏟n fois=n\displaystyle{1 + q^1 + q^2 + \dots + q^n = \underbrace{1 + 1 + 1 + \dots + 1}_{n \text{ fois}} = n}1+q1+q2+⋯+qn=n fois1+1+1+⋯+1​​=n.

Étude des suites

Sens de variation

Définition

Soit (un)(u_n)(un​) une suite.

  • (un)(u_n)(un​) est croissante si on a un+1≥unu_{n+1} \geq u_nun+1​≥un​ (ou un+1−un≥0u_{n+1} - u_n \geq 0un+1​−un​≥0) pour tout n∈Nn \in \mathbb{N}n∈N.

  • (un)(u_n)(un​) est décroissante si on a un+1≤unu_{n+1} \leq u_nun+1​≤un​ (ou un+1−un≤0u_{n+1} - u_n \leq 0un+1​−un​≤0) pour tout n∈Nn \in \mathbb{N}n∈N.

  • (un)(u_n)(un​) est dite constante s’il existe c∈Rc \in \mathbb{R}c∈R tel que un=cu_n = cun​=c pour tout n∈Nn \in \mathbb{N}n∈N.

Si une suite est croissante ou décroissante et ne change pas de variation, alors elle est dite monotone.

Introduction aux limites

Quand on souhaite s’intéresser à la limite d’une suite (un)(u_n)(un​), on étudie le comportement de ses termes quandnnn devient très grand. On préfère dire alors que nnn tend vers +∞+\infty+∞.

Définition

Soit (un)(u_n)(un​) une suite.

  • Si (un)(u_n)(un​) tend vers un réel quand nnn tend vers +∞+\infty+∞, on dit qu’elle converge.

  • Si (un)(u_n)(un​) tend vers une limite infinie quand nnn tend vers +∞+\infty+∞, on dit qu’elle diverge.

À savoir que si une suite a une limite, alors cette limite est unique. Mais il est également possible pour une suite de ne pas admettre de limite.

Représentation graphique

Il est possible de représenter graphiquement une suite. Cela peut aider, par exemple dans le but de chercher sa limite.

Méthode pour une suite définie par récurrence

Soit (un)(u_n)(un​) une suite définie par récurrence. Pour représenter (un)(u_n)(un​) dans un graphique :

  1. On trace la droite d’équation y=xy = xy=x.

  2. Comme cette suite est définie par récurrence, pour tout entier nnn on a une relation du type un+1=f(un)u_{n+1} = f(u_n)un+1​=f(un​). Il s’agit de tracer la courbe représentative Cf\mathcal{C}_fCf​ de la fonction fff.

  3. On place le point AAA de coordonnées (u0;0)(u_0; 0)(u0​;0)

  4. On trace une droite verticale passant par AAA, son intersection avec Cf\mathcal{C}_fCf​ donne un point B=(u0;u1)B = (u_0; u_1)B=(u0​;u1​).

  5. À l’aide du point BBB, on place le point C=(0;u1)C = (0; u_1)C=(0;u1​).

  6. On trace une droite horizontale passant par CCC, son intersection avec la droite y=xy = xy=x donne un point D=(u1;u1)D = (u_1; u_1)D=(u1​;u1​).

  7. Une fois le point DDD obtenu, on place le point (u1;0)(u_1; 0)(u1​;0).

  8. On recommence l’opération en remplaçant u0u_0u0​ par u1u_1u1​ et u1u_1u1​ par u2u_2u2​, puis on recommence, etc.

Il est cependant plus facile de représenter graphiquement une suite dont on connaît le terme général.

Méthode pour une suite définie par son terme général

Soit (vn)(v_n)(vn​) une suite définie par son terme général. Pour représenter (vn)(v_n)(vn​) dans un graphique :

  1. On place le point de coordonnées (0;v0)(0; v_0)(0;v0​).

  2. On place le point de coordonnées (1;v1)(1; v_1)(1;v1​).

  3. On place le point de coordonnées (2;v2)(2; v_2)(2;v2​). Etc.

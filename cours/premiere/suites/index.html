Chapitre I – Les suites

Première Difficile

En mathématiques, une suite est une famille d'éléments — appelés ses termes — indexée par les entiers naturels. Ce cours va donc vous apprendre comment définir une suite et l'étudier. On y verra notamment l'étude du sens de variation, la représentation de suites dans le plan, et plus encore !

Lire le résumé

Généralités

Définition

On appelle suite une fonction de N\mathbb{N}N dans R\mathbb{R}R : cette fonction va prendre des éléments de l’ensemble de départ N\mathbb{N}N et va les amener dans l’ensemble d’arrivée R\mathbb{R}R.

Définition

Il y a plusieurs manières de définir une suite :

  • Par récurrence : On donne le premier terme de la suite ainsi que le terme au rang n+1n+1n+1.

  • Par son terme général : On donne le nnn-ième terme de la suite en fonction de nnn.

Attention ! Bien que ces deux modes de génération soient les principaux, il en existe d’autres : algorithme, motifs géométriques, ...

Exemple

On définit les suites (un)(u_n)(un​) et (vn)(v_n)(vn​) ainsi :

  • un=nu_n = nun​=n pour tout n∈Nn \in \mathbb{N}n∈N ((un)(u_n)(un​) est définie par son terme général).

  • (vn)={v0=0vn+1=vn+1 pour tout n≥1(v_n) = \begin{cases} v_0 = 0 \\ v_{n+1} = v_n + 1 \text{ pour tout } n \geq 1 \end{cases}(vn​)={v0​=0vn+1​=vn​+1 pour tout n≥1​ ((vn)(v_n)(vn​) est définie par récurrence).

On remarque que bien que définies différemment, (un)(u_n)(un​) et (vn)(v_n)(vn​) sont égales.

À ne pas confondre :

  • (un)(u_n)(un​) qui est la suite (un)(u_n)(un​).

  • unu_nun​ qui est le nnn-ième terme de la suite (un)(u_n)(un​).

Ce ne sont pas les mêmes objets : le premier est une suite, le second est un réel.

Suites arithmétiques

Définition

Une suite (un)(u_n)(un​) est dite arithmétique si elle est de la forme un+1=un+ru_{n+1} = u_n + run+1​=un​+r avec r∈Rr \in \mathbb{R}r∈R.

Raison

Le réel rrr est la raison de la suite (si r>0r > 0r>0, (un)(u_n)(un​) est strictement croissante, si r<0r < 0r<0, (un)(u_n)(un​) est strictement décroissante et si r=0r = 0r=0, (un)(u_n)(un​) est constante).

Il est possible de trouver le terme général d’une suite arithmétique :

Terme général

On note ppp le rang initial de la suite (celui à partir duquel la suite est définie). Alors, pour tout n≥pn \geq pn≥p : un=up+(n−p)×ru_n = u_p + (n-p) \times run​=up​+(n−p)×r Et si (un)(u_n)(un​) est définie à partir du rang 000 (on a p=0p = 0p=0), alors pour tout n∈Nn \in \mathbb{N}n∈N : un=u0+(n−0)×r=u0+n×ru_n = u_0 + (n-0) \times r = u_0 + n \times run​=u0​+(n−0)×r=u0​+n×r

Démonstration

Terme général

On a up+1=up+ru_{p+1} = u_p + rup+1​=up​+r. Puis, up+2=up+1+r=up+r+r=up+2×ru_{p+2} = u_{p+1} + r = u_p + r + r = u_p + 2 \times rup+2​=up+1​+r=up​+r+r=up​+2×r. De même, up+3=up+2+r=up+3×ru_{p+3} = u_{p+2} + r = u_p + 3 \times rup+3​=up+2​+r=up​+3×r et caetera. En fait, pour tout kkk entier plus grand que ppp, on a up+k=up+k×ru_{p+k} = u_p + k \times rup+k​=up​+k×r. Donc si on pose n=p+kn = p+kn=p+k, alors un=up+(n−p)×ru_n = u_p + (n-p) \times run​=up​+(n−p)×r.

Somme des termes

Pour tout n∈N∗n \in \mathbb{N}^*n∈N∗, 1+2+⋯+n=n(n+1)21 + 2 + \dots + n = \frac{n(n + 1)}{2}1+2+⋯+n=2n(n+1)​

Démonstration

Somme des termes

On pose pour tout n∈Nn \in \mathbb{N}n∈N, Sn=1+2+⋯+nS_n = 1 + 2 + \dots + nSn​=1+2+⋯+n. On a également Sn=n+(n−1)+⋯+1S_n = n + (n-1) + \dots + 1Sn​=n+(n−1)+⋯+1 (en écrivant la somme à l’envers). D’où Sn+Sn=2Sn=(n+1)+(n+1)+⋯+(n+1)⏟n fois=n×(n+1)S_n + S_n = 2S_n = \underbrace{(n + 1) + (n + 1) + \dots + (n + 1)}_{n \text{ fois}} = n \times (n + 1)Sn​+Sn​=2Sn​=n fois(n+1)+(n+1)+⋯+(n+1)​​=n×(n+1). Et ainsi Sn=n(n+1)2S_n = \frac{n(n + 1)}{2}Sn​=2n(n+1)​.

Exemple

On souhaite calculer S=24+25+⋯+104S = 24 + 25 + \dots + 104S=24+25+⋯+104.

En fait, S=1+2+⋯+23+24+25+⋯+104−(1+2+⋯+23)S = 1 + 2 + \dots + 23 + 24 + 25 + \dots + 104 - (1 + 2 + \dots + 23)S=1+2+⋯+23+24+25+⋯+104−(1+2+⋯+23). Calculons les deux sommes séparément :

  • 1+2+⋯+23=23×242=2761 + 2 + \dots + 23 = \frac{23 \times 24}{2} = 2761+2+⋯+23=223×24​=276

  • 1+2+⋯+104=104×1052=54601 + 2 + \dots + 104 = \frac{104 \times 105}{2} = 54601+2+⋯+104=2104×105​=5460

D’où S=5460−276=5184S = 5460 - 276 = 5184S=5460−276=5184.

Suites géométriques

Définition

Une suite (vn)(v_n)(vn​) est dite géométrique si elle est de la forme vn+1=vn×qv_{n+1} = v_n \times qvn+1​=vn​×q avec q∈Rq \in \mathbb{R}q∈R.

Raison

Le réel qqq est la raison de la suite (si q>1q > 1q>1, (vn)(v_n)(vn​) est strictement croissante, si 0<q<10 < q < 10<q<1, (vn)(v_n)(vn​) est strictement décroissante et si q=1q = 1q=1 ou 000, (vn)(v_n)(vn​) est constante).

Il est possible de trouver le terme général d’une suite géométrique :

Terme général

On note ppp le rang initial de la suite (celui à partir duquel la suite est définie). Alors, pour tout n≥pn \geq pn≥p : vn=vp×qn−pv_n = v_p \times q^{n-p}vn​=vp​×qn−p Et si (un)(u_n)(un​) est définie à partir du rang 000 (on a p=0p = 0p=0), alors pour tout n∈Nn \in \mathbb{N}n∈N : vn=v0×qn−0=v0×qnv_n = v_0 \times q^{n-0} = v_0 \times q^nvn​=v0​×qn−0=v0​×qn

Démonstration

Terme général

On a vp+1=vp×qv_{p+1} = v_p \times qvp+1​=vp​×q. Puis, vp+2=vp+1×q=vp×q×q=vp×q2v_{p+2} = v_{p+1} \times q = v_p \times q \times q = v_p \times q^2vp+2​=vp+1​×q=vp​×q×q=vp​×q2. De même, vp+3=vp+2×q=vp×q3v_{p+3} = v_{p+2} \times q = v_p \times q^3vp+3​=vp+2​×q=vp​×q3 et caetera. En fait, pour tout kkk entier plus grand que ppp, on a vp+k=vp×qkv_{p+k} = v_p \times q^kvp+k​=vp​×qk. Donc si on pose n=p+kn = p+kn=p+k, alors vn=vp×qn−pv_n = v_p \times q^{n-p}vn​=vp​×qn−p.

Somme des termes

Soit n≠0n \neq 0n=0 un entier et qqq un réel, alors :

  • Si q≠1q \neq 1q=1, alors 1+q1+q2+⋯+qn=1−qn+11−q1 + q^1 + q^2 + \dots + q^n = \frac{1 - q^{n + 1}}{1 - q}1+q1+q2+⋯+qn=1−q1−qn+1​.

  • Si q=1q = 1q=1, alors 1+q1+q2+⋯+qn=1+1+1+⋯+1⏟n fois=n1 + q^1 + q^2 + \dots + q^n = \underbrace{1 + 1 + 1 + \dots + 1}_{n \text{ fois}} = n1+q1+q2+⋯+qn=n fois1+1+1+⋯+1​​=n.

Démonstration

Somme des termes

Le cas q=1q = 1q=1 étant donné juste au-dessus, on supposera q≠1q \neq 1q=1. On pose pour tout n∈Nn \in \mathbb{N}n∈N, Sn=1+q1+q2+⋯+qnS_n = 1 + q^1 + q^2 + \dots + q^nSn​=1+q1+q2+⋯+qn.

On a : qSn=q1+q2+q3+⋯+qn+1qS_n = q^1 + q^2 + q^3 + \dots + q^{n+1}qSn​=q1+q2+q3+⋯+qn+1, puis : Sn−qSn=1+q1+q2+⋯+qn−q1−q2−q3−⋯−qn+1=1−qn+1S_n - qS_n = 1 + q^1 + q^2 + \dots + q^n - q^1 - q^2 - q^3 - \dots - q^{n+1} = 1 - q^{n+1}Sn​−qSn​=1+q1+q2+⋯+qn−q1−q2−q3−⋯−qn+1=1−qn+1.

Donc on a en factorisant par SnS_nSn​ : (1−q)Sn=1−qn+1  ⟺  Sn=1−qn+11−q(1 - q)S_n = 1 - q^{n+1} \iff S_n = \frac{1 - q^{n+1}}{1 - q}(1−q)Sn​=1−qn+1⟺Sn​=1−q1−qn+1​.

Exemple

On souhaite calculer S=35+36+⋯+310S = 3^5 + 3^6 + \dots + 3^{10}S=35+36+⋯+310.

En fait, S=1+3+⋯+34+35+36+⋯+310−(1+⋯+34)S = 1 + 3 + \dots + 3^4 + 3^5 + 3^6 + \dots + 3^{10} - (1 + \dots + 3^4)S=1+3+⋯+34+35+36+⋯+310−(1+⋯+34). Calculons les deux sommes séparément :

  • 1+3+⋯+34=1−351−3=1211 + 3 + \dots + 3^4 = \frac{1 - 3^5}{1 - 3} = 1211+3+⋯+34=1−31−35​=121

  • 1+3+⋯+310=1−3111−3=885731 + 3 + \dots + 3^{10} = \frac{1 - 3^{11}}{1 - 3} = 885731+3+⋯+310=1−31−311​=88573

D’où S=88573−121=88452S = 88573 - 121 = 88452S=88573−121=88452.

Étude des suites

Sens de variation

Définition

Soit (un)(u_n)(un​) une suite.

  • (un)(u_n)(un​) est croissante si on a un+1≥unu_{n+1} \geq u_nun+1​≥un​ (ou un+1−un≥0u_{n+1} - u_n \geq 0un+1​−un​≥0) pour tout n∈Nn \in \mathbb{N}n∈N.

  • (un)(u_n)(un​) est décroissante si on a un+1≤unu_{n+1} \leq u_nun+1​≤un​ (ou un+1−un≤0u_{n+1} - u_n \leq 0un+1​−un​≤0) pour tout n∈Nn \in \mathbb{N}n∈N.

  • (un)(u_n)(un​) est dite constante s’il existe c∈Rc \in \mathbb{R}c∈R tel que un=cu_n = cun​=c pour tout n∈Nn \in \mathbb{N}n∈N.

Si une suite est croissante ou décroissante et ne change pas de variation, alors elle est dite monotone.

Introduction aux limites

Quand on souhaite s’intéresser à la limite d’une suite (un)(u_n)(un​), on étudie le comportement de ses termes quandnnn devient très grand. On préfère dire alors que nnn tend vers +∞+\infty+∞.

Définition

Soit (un)(u_n)(un​) une suite.

  • Si (un)(u_n)(un​) tend vers un réel quand nnn tend vers +∞+\infty+∞, on dit qu’elle converge.

  • Si (un)(u_n)(un​) tend vers une limite infinie quand nnn tend vers +∞+\infty+∞, on dit qu’elle diverge.

Exemple

On définit la suite (un)(u_n)(un​) pour tout n∈Nn \in \mathbb{N}n∈N par un=1nu_n = \frac{1}{n}un​=n1​. On souhaite trouver la limite possible de cette suite en +∞+ \infty+∞. Pour cela, regardons les valeurs que prend cette suite pour des valeurs de nnn très grandes :

100100100 0,010,010,01
10001 0001000 0,0010,0010,001
100000100 000100000 0,000010,000 010,00001
10000000001 000 000 0001000000000 0,0000000010,000 000 0010,000000001

Il semble que cette suite converge vers 0.

À savoir que si une suite a une limite, alors cette limite est unique. Mais il est également possible pour une suite de ne pas admettre de limite.

Exemple

On définit la suite (un)(u_n)(un​) pour tout n∈Nn \in \mathbb{N}n∈N par un=(−1)nu_n = (-1)^nun​=(−1)n. On souhaite trouver la limite possible de cette suite en +∞+ \infty+∞.

100100100 111
101101101 −1-1−1
10000001 000 0001000000 111
10000011 000 0011000001 −1-1−1

En fait, si nnn est pair cette suite vaut 111 et si nnn est impair elle vaut −1-1−1. Cette suite n’admet donc pas de limite : elle diverge.

Représentation graphique

Il est possible de représenter graphiquement une suite. Cela peut aider, par exemple dans le but de chercher sa limite.

Méthode pour une suite définie par récurrence

Soit (un)(u_n)(un​) une suite définie par récurrence. Pour représenter (un)(u_n)(un​) dans un graphique :

  1. On trace la droite d’équation y=xy = xy=x.

  2. Comme cette suite est définie par récurrence, pour tout entier nnn on a une relation du type un+1=f(un)u_{n+1} = f(u_n)un+1​=f(un​). Il s’agit de tracer la courbe représentative Cf\mathcal{C}_fCf​ de la fonction fff.

  3. On place le point AAA de coordonnées (u0;0)(u_0; 0)(u0​;0)

  4. On trace une droite verticale passant par AAA, son intersection avec Cf\mathcal{C}_fCf​ donne un point B=(u0;u1)B = (u_0; u_1)B=(u0​;u1​).

  5. À l’aide du point BBB, on place le point C=(0;u1)C = (0; u_1)C=(0;u1​).

  6. On trace une droite horizontale passant par CCC, son intersection avec la droite y=xy = xy=x donne un point D=(u1;u1)D = (u_1; u_1)D=(u1​;u1​).

  7. Une fois le point DDD obtenu, on place le point (u1;0)(u_1; 0)(u1​;0).

  8. On recommence l’opération en remplaçant u0u_0u0​ par u1u_1u1​ et u1u_1u1​ par u2u_2u2​, puis on recommence, etc.

Exemple

Représentation des trois premiers termes de la suite (un)={u0=3un+1=un2(u_n) = \begin{cases} u_0 = 3 \\ u_{n+1} = \frac{u_n}{2} \end{cases}(un​)={u0​=3un+1​=2un​​​.

Il est cependant plus facile de représenter graphiquement une suite dont on connaît le terme général.

Méthode pour une suite définie par son terme général

Soit (vn)(v_n)(vn​) une suite définie par son terme général. Pour représenter (vn)(v_n)(vn​) dans un graphique :

  1. On place le point de coordonnées (0;v0)(0; v_0)(0;v0​).

  2. On place le point de coordonnées (1;v1)(1; v_1)(1;v1​).

  3. On place le point de coordonnées (2;v2)(2; v_2)(2;v2​). Etc.

Exemple

Représentation des trois premiers termes de la suite (vn)(v_n)(vn​) définie pour tout n∈Nn \in \mathbb{N}n∈N par vn=2nv_n = 2^nvn​=2n.
